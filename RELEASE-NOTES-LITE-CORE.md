# Lucky Arcade Lite Core 1.0.0

Lite Core is a product reset, not another feature-heavy release.

## Core experience

- Four games: Dice, European Roulette, Coin Flip and Lucky Slots
- Compact game lobby and simplified navigation
- Dedicated play and wallet history
- Cleaner account, fairness and admin pages
- Responsive dark arcade design with restrained micro-interactions

## Core integrity retained

- PostgreSQL production target
- Redis cache and sessions
- Database transactions and row locks
- Idempotent game requests
- Immutable ledger records
- Versioned rulesets and fairness checks
- Admin audit trail, backup and wallet reconciliation
- TOTP security

## Removed

Gamification, referrals, promos, tickets, leagues, live polling, presence, announcements, detailed analytics, simulation UI and complex RBAC.
