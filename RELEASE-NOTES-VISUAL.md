# Lucky Arcade Visual 1.1.0

Visual 1.1.0 is a presentation upgrade over Lite Core. It deliberately does not add database tables, reward systems or additional games.

## What changed

- New visual identity and local SVG artwork
- Animated game lobby and ambient background
- Numeric dice cube animation
- European roulette wheel animation
- Double-sided coin animation
- Sequential slot reel animation
- Optional generated sounds
- Win celebration
- Reduced-motion mode and system preference support
- Mobile-responsive control panels

## What did not change

- Game engines and payout rules
- HMAC-SHA256 result generation
- Wallet transaction boundaries
- Ledger reconciliation
- Request idempotency
- Admin permissions
- Database schema

The animation always displays a result already generated and stored by Laravel.
