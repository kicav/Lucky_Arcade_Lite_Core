# Next edition: Lucky Arcade Visual

Lite Core intentionally keeps animation light. The next edition can replace only the presentation layer while leaving wallet, game engines and database contracts unchanged.

Planned visual work:

- Illustrated lobby and per-game backgrounds
- CSS/Canvas dice and coin motion
- Animated roulette wheel and ball
- Sequential slot reels and win particles
- Optional sound, mute and reduced-motion controls
- Mobile performance mode and preloaded optimized assets

The backend must settle a result first; animation only presents that result.
